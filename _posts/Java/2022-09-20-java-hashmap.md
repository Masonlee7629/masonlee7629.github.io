## HashMap
#### 스레드(Thread)
프로세스는 실행 중인 애플리케이션을 의미한다.<br>
프로세스는 데이터, 컴퓨터 자원, 스레드로 구성된다.<br>

여기서 스레드는 데이터와 애플리케이션이 확보한 자원을 활용하여 소스코드를 실행한다.<br>
즉, 스레드는 하나의 코드 실행 흐름이다.<br>


스레드를 하나만 가지는 프로세스를 싱글 스레드 프로세스, 여러 개의 스레드를 가진 프로세스를 멀티 스레드 프로세스라고 한다.<br>

여기서 멀티 스레드 프로세스는 해당 애플리케이션이 동시 작업을 할 수 있다는 것을 의미한다.<br>
그리고 멀티 스레드는 다수의 클라이언트 요청을 처리하는 서버를 개발할 때 사용된다.

#### 자바 가상 머신(Java Virtual Machine)
JVM은 자바 프로그램을 실행시키는 도구로 자바로 작성한 소스 코드를 해석하여 실행하는 프로그램이다.<br>
각 운영체제에 적합한 JVM이 따로 존재하며 JVM은 자바 소스 코드를 운영 체제에 맞게 변환해 실행시켜 자바가 
운영체제로부터 독립적으로 동작할 수 있도록 한다.

자바에는 가비지 컬렉션(Garbage Collection)이라는 메모리를 자동으로 관리하는 프로세스가 포함되어 있다.<br>
가비지 컬렉션은 프로그램에서 더 이상 사용하지 않는 객체를 찾아 삭제하거나 제거하여 메모리를 확보하는 것을 의미한다.
