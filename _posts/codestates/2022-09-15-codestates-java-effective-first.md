### Java 심화(Effective)
#### 애너테이션(Annotation)
자바를 공부하며 주석(Comment)을 종종 사용해왔다.<br>
주석은 개발자의 입장에서 다음과 같은 역할을 수행한다.
- 스스로 좀 더 직관적이고 빠르게 코드를 이해할 수 있도록 지원하는 역할
- 다른 개발자와의 협업의 과정에서 해당 코드를 설명하는 정보를 제공하는 역할

애너테이션은 주석과 기능적으로 같은 역할을 담당한다고 할 수 있으며 정보를 전달하는 대상에 주요한 차이가 있다고 할 수 있다.
- 주석 : 소스 코드를 읽는 '사람'에게 정보를 제공
- 애너테이션 : 특정 코드를 사용하는 '프로그램'에게 정보를 전달

#### 람다(Lambda)
람다식은 함수형 프로그래밍 기법은 지원하는 자바의 문법요소이다.<br>
메서드를 하나의 '식(expression)'으로 표현한 것으로 코드를 매우 간결하고 명확하게 표현할 수 있다.<br>
기본적으로 반환타입과 이름을 생략할 수 있기에 이를 익명 함수(anonymous function)이라고 부른다.

#### 스트림(Stream)
스트림은 배열, 컬렉션의 저장 요소를 하나씩 참조해서 람다식으로 처리할 수 있도록 해주는 반복자이다.<br>
스트림을 사용하면 List, Set, Map 배열 등 다양한 데이터 소스로부터 스트림을 만들 수 있고 표준화된 방법으로 다룰 수 있다.<br>
스트림의 주요 특징을 다음과 같다.
1. 선언형으로 데이터 소스를 처리
2. 람다식으로 요소 처리 코드를 제공
3. 내부 반복차를 사용하여 병렬 처리가 용이
4. 중간 연산과 최종 연산 가능

#### 파일 입출력(I/O)
자바에서는 파일 입출력을 다루기 위한 스트림을 제공한다.
1. 바이트 기반 스트림 : InputStream, OutputStream
2. 문자 기반 스트림 : FileReader, FileWriter
